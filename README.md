# AI Career Guidance Platform

An AI-powered career recommendation platform built using HTML, CSS, and JavaScript.

## Features
- Career Recommendations
- Career Match Score
- Skill Gap Analysis
- Learning Roadmap
- Certifications
- Interview Questions
- Project Recommendations
- Top Hiring Companies
- PDF Report Download

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- index.html
- style.css
- script.js
